ACM Tutorials

Introducing you to computing concepts not taught in class.

Programming Language Tutorials. Throughout the school year we offer various tutorials on programming languages that are not covered in classes as well as general skills necessary in the modern programming workforce. From the basics of using version control such as Git to an introduction on web programming, our programming tutorials offer something for programming beginners and the knowledgeble alike.

Game Development Tutorials. Every semester we hold a two part game development series. After this we put your knowledge to the test with a forty-eight hour Game Jam where we see what you you can come up with! Snacks and drinks are provided during this time.

Cyber Security Tutorials.  Recently added are cyber security based tutorials. Topics discussed are pentesting, For legal reasoning those attending our tutorials will have to sign a legal waiver. 

IEEE Tutorials

Offering hands-on tutorials for computer engineering concepts.

Microcontroller Development with Arduino. Programming for microcontrollers can have certain unique requirements compared to programming for desktop or laptop computers. During the semseter we will offer several tutorials on programming for Arduinos, and will cover various basic Arduino projects.

Parametric 3D CAD with OpenSCAD. Being able to model objects in three dimensions is a major step toward being able to manufacture your own parts. We will be offering tutorials using OpenSCAD to teach basic CAD concepts and techniques.

Fused Filimanet Fabrication Introduction While choosing parameters for a 3D print can seem like a daunting task at first, we will offer tutorials that explain what the various parameters mean and how they effect the overall print.
